Conhecimentos

HTML é a linguagem usada para estruturar páginas da web. Ela define elementos como títulos, textos, imagens, links e formulários que aparecem em um site.

CSS é a linguagem usada para estilizar páginas da web. Ele define cores, fontes, tamanhos, espaçamentos e layout dos elementos do HTML.

JavaScript é a linguagem usada para adicionar interatividade às páginas web, como cliques, animações, validações e atualizações de conteúdo.

Python é uma linguagem de programação usada para criar sistemas, automações, sites, análise de dados e inteligência artificial.

Bootstrap é um framework de CSS usado para criar sites responsivos e modernos mais rápido, com componentes prontos como botões, grids, menus e formulários.

Tailwind CSS é um framework de CSS que permite estilizar páginas diretamente nas classes do HTML, facilitando criar interfaces rápidas, modernas e responsivas.

MySQL é um sistema de banco de dados usado para armazenar, organizar e gerenciardados de aplicações e sites.
